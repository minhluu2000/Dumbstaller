The full notation example:

	Host name: someone
	IP address: 192.168.1.69
	Subnet Mask: 255.255.255.0
	Default gateway: 192.168.1.1
	
	Preferred DNS server: 8.8.8.8
	Alternate DNS server: 1.1.1.1

The one to be imported to python:
	0      , 1          , 2           , 3         , 4     , 5
	host   , ip ad      , submask     , def gw    , dns_1 , dns_2

	someone,192.168.1.69,255.255.255.0,192.168.1.1,8.8.8.8,1.1.1.1


Note: There should be no space for the imported format.
